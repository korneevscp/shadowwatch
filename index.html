Fonctionnalités

Tout ce dont tu as besoin,
rien de superflu

Un outil Python léger, sans dépendances lourdes, qui tourne en arrière-plan et t'alerte dès qu'il trouve quelque chose.

🔍
Surveillance multi-sources
Google, Pastebin, BreachDirectory, Ahmia (index Tor). Chaque keyword scanné sur toutes les sources à chaque exécution.
📧
Alerte email systématique
Un email à chaque scan — qu'il y ait des résultats ou non. Sujet coloré selon le statut : [ALERTE] ou [OK].
🌐
Dashboard web local
Interface sur localhost:8080 avec historique des 50 derniers scans, stats et bouton de scan manuel.
⏰
Automatisation quotidienne
Un fichier .bat installe la tâche Windows en un clic. Le scan se lance chaque jour à 09h00 sans intervention.
🔒
100% local & privé
Aucune donnée envoyée à un tiers. Tout tourne sur ta machine. Tes keywords restent dans ton fichier Python.
💸
Entièrement gratuit
Aucun abonnement, aucune API payante. Utilise uniquement des sources et bibliothèques open-source.

Installation

Prêt en 5 minutes

Python 3.8+ requis. Fonctionne sur Windows, Mac et Linux.

1

Cloner le dépôt

Télécharge les fichiers depuis GitHub ou clone le projet.

git clone https://github.com/korneevscp/shadowwatch
2

Installer les dépendances

Une seule commande pour tout installer.

pip install requests beautifulsoup4 googlesearch-python
3

Configurer monitor.py

Renseigne ton email Gmail, ton App Password Google, et tes keywords à surveiller dans les premières lignes du fichier.

4

Lancer (Windows)

Double-clique sur installer.bat en tant qu'administrateur pour installer la tâche quotidienne automatique.

py monitor.py  |  py dashboard.py
Limitations

Ce qu'il faut savoir
avant d'utiliser

ShadowWatch fonctionne avec des sources publiques gratuites. Certaines ont des restrictions à connaître.

⚠️

Google 429 — Blocage si relance trop rapide

Google détecte les scripts automatiques et peut bloquer temporairement les requêtes avec une erreur 429 Too Many Requests. Si cela arrive :

  1. Attends 30 minutes avant de relancer le script
  2. Le script intègre déjà des pauses automatiques (15s entre requêtes, 30s entre keywords)
  3. Si le problème persiste, augmente les valeurs sleep_interval et time.sleep() dans monitor.py
⚖️

Usage personnel uniquement

Ce projet est conçu pour surveiller vos propres données personnelles. Surveiller les informations d'autrui sans consentement est illégal (RGPD, Art. 226-1 du Code pénal français).

📶

Connexion internet requise

Le scan nécessite une connexion active. La tâche automatique échouera silencieusement si le PC est éteint ou hors ligne à l'heure planifiée — elle se relancera le lendemain.
